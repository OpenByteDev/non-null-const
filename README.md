This crate provides [`NonNullConst`](https://docs.rs/non-null-const/latest/non-null-const/struct.NonNullConst.html), a **non-null, covariant** raw **const** pointer type. It is conceptually the `*const` analogue of [`core::ptr::NonNull`](https://doc.rust-lang.org/std/ptr/struct.NonNull.html).
